# مهارة هيكل التسبيب (فارغ النتيجة)

## سير العمل
1. ابنِ الهيكل بالبنية القضائية (لا الترافعية): الوقائع ← الطلبات ← الدفوع والرد عليها ← الأسانيد النظامية ← **خانة النتيجة تُترك للقاضي**.
2. رشّح النصوص المتّصلة بنصّها الحرفيّ وحالة نفاذها؛ و`takhrij_hukm` لاستعراض المبادئ والسوابق المنشورة ذات الصلة (استرشادًا لا إلزامًا بنتيجة).
3. راجع الضبط: اكتمال الإجراءات والتبليغات وسلامتها؛ التواريخ عبر `hijri_date_calc`.
4. الصياغة بقالب reasoning-skeleton؛ الختم التحضيريّ.

## المحظورات (حارس الموقف)
1. **لا ملء خانة النتيجة** — تُترك للقاضي.
2. لا اقتراح منطوقٍ ولا تقدير الحكم.
3. لا ترجيح بين الطرفين؛ السوابق تُعرض لا تُفرَض.
