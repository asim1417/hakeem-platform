2. 🔴 **بيانات:** استيراد أحكام حقيقية (`import:judgments`) — لا مصدر في المستودع.
3. 🟠 تعبئة المتجهات + تفعيل البحث الدلالي (`backfill-embeddings` + `SEMANTIC_SEARCH=true`).
4. 🟠 لوحة إدارة محتوى (إضافة/تعديل نظام/مادة + إعادة فهرسة).
5. 🟡 Pagination + Caching + تلميع أداتي الاختبار + إطار القاضي.

## 5) أوامر التشغيل
```bash
npm install
docker compose up -d            # أو PostgreSQL محلي + pgvector
npm run db:push
npm run db:seed                 # 9 أنظمة + 1981 مادة + أدوار/مستخدم إداري
npm run seed:demo-judgments     # عيّنة أحكام موسومة (اختياري)
npm run db:search-indexes       # فهارس trigram
npm run build && npm start
# تفعيل الذكاء: /admin/ai → اختر Anthropic + مفتاح + claude-sonnet-4-6
```
