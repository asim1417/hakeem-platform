## خطة المراحل الأربع

**المرحلة ١ — أساس واستقرار (أسبوع)**: P0 + تفكيك GameScene + نشر dist منفصل + خط Cairo.
**المرحلة ٢ — إحساس وتحكم (أسبوع)**: كاميرات (زوم/فيد/سلو-مو) + قائم وعارضة + دفاع بالإصبع في المباراة + إصلاح سهم التصويب تحت الإصبع.
**المرحلة ٣ — جرافيكس وهوية (أسبوعان)**: حارس sprite كامل + ملاعب سعودية + أيقونات موحدة + شعار + مصغرات الخزنة + WebP.
**المرحلة ٤ — محتوى وصقل (أسبوعان)**: الفاولات + تحدي يومي + لعب محلي + أصوات بشرية + شجرة بطولة + مودالات + شاشة تتويج فاخرة.

الناتج بعد ~٦ أسابيع: لعبة بلنتيات عربية للأطفال بمستوى منافس تجاريًا على الويب.
