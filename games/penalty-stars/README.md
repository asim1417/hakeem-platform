# ⭐ نجوم البلنتيات ⚽

لعبة ضربات جزاء بسيطة وممتعة للأطفال (٦–٧ سنوات) — تعمل على المتصفح والجوال.

## التقنيات
- **Phaser 3** + **TypeScript** — محرك اللعبة والفيزياء (Arcade Physics)
- **Vite** — تشغيل وبناء سريع
- **Howler.js** — المؤثرات الصوتية (مولّدة برمجيًا، بلا ملفات خارجية)
- **GSAP** — الحركات الاحتفالية والانتقالات

## التشغيل

```bash
cd games/penalty-stars
npm install
npm run dev      # تشغيل محلي على http://localhost:5173
npm run build    # بناء نسخة الإنتاج (ملف HTML واحد في dist/)
```

نسخة الإنتاج ملف HTML واحد مستقل (`dist/index.html`) — يمكن نشره على أي استضافة ثابتة أو فتحه مباشرة.

## طريقة اللعب
1. اختر لاعبك المفضل من ٧ شخصيات مرحة.
2. اختر مستوى الحارس: سهل / متوسط / بطل.
3. اسحب من الكرة نحو زاوية المرمى — طول السحب يحدد القوة.
4. لديك ٥ تسديدات، وكل هدف يمنحك نجمة ⭐.
5. سجّل ٣ أهداف أو أكثر لتصبح **نجم البلنتيات اليوم!**

## مبادئ التصميم للأطفال
- لا خسارة قاسية ولا عبارات سلبية — تشجيع دائم.
- أزرار كبيرة وألوان مبهجة وخط واضح.
- لا إعلانات، لا مشتريات، لا جمع بيانات، لا دردشة.

## استبدال الأصول المؤقتة
الأشكال الحالية مرسومة برمجيًا في `src/scenes/BootScene.ts`.
لاستخدام صور حقيقية: ضع ملفات PNG في `public/assets/images/`
وحمّلها في `BootScene` بـ `this.load.image('ball', 'assets/images/ball.png')`
بنفس أسماء المفاتيح (`ball`, `star`, `keeper`, `avatar-{id}`).
الأصوات في `src/utils/audio.ts` قابلة للاستبدال بملفات mp3 في `public/assets/audio/`.

## هيكل الملفات
```
src/
├── main.ts                     نقطة الدخول وإعداد Phaser
├── styles.css
├── config/gameConfig.ts        الأبعاد والألوان والصعوبة والعبارات
├── data/players.ts             بيانات اللاعبين السبعة
├── scenes/
│   ├── BootScene.ts            توليد الأصول
│   ├── MenuScene.ts            واجهة البداية
│   ├── PlayerSelectScene.ts    اختيار الشخصية
│   ├── GameScene.ts            اللعب: السحب والتسديد والحارس
│   └── ResultScene.ts          النتيجة بالنجوم
└── utils/
    ├── audio.ts                Howler + أصوات مولّدة
    ├── animations.ts           GSAP: نجوم وقصاصات واحتفالات
    └── ui.ts                   أزرار كبيرة للأطفال
```
