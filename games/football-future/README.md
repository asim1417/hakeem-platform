# ⚡ فوتبول فيوتشر — Football Future

> اِلعب. تطوّر. انتصر. — *PLAY. GROW. WIN.*

لعبة كرة قدم عربية الهوية للناشئين (7–14 سنة)، مبنية من **حزمة الكود v3** المسلَّمة مع
حزمة الهوية البصرية، مع ترقية جرافيكس شاملة لمستوى اللوحات المرجعية الست.

## الرابط

اللعبة **مستقلة تماماً عن واجهة منصة حكيم** وتُقدَّم على مسار خاص بها:

```
https://<domain>/football-future
```

المسار معرَّف كـ rewrite في `next.config.mjs`، والملفات ثابتة تُخدم من `public/football-future/`
— لا تشارك أي layout أو أصول مع المنصة، وتعمل كتطبيق PWA قابل للتثبيت.

## بنية الكود (من حزمة v3)

```
public/football-future/
├── index.html                  الهيكل + خطوط الهوية (Noto Kufi Arabic + Rajdhani)
├── manifest.webmanifest        PWA — عرضي، أيقونة، نطاق /football-future/
├── service-worker.js           عمل دون اتصال (cache-first)
└── src/
    ├── css/tokens.css          ألوان وخطوط ومسافات الهوية (DESIGN_TOKENS)
    ├── css/styles.css          نظام الواجهة: زجاجية داكنة، أزرار مائلة، توهجات
    ├── js/data.js              بيانات الفريق والمهام والمتجر والنصوص العربية
    ├── js/audio.js             مؤثرات WAV أصلية + مولّد نغمات + صوت عربي (SpeechSynthesis)
    ├── js/commentary.js        معلّق عربي ببنك عبارات حسب أحداث المباراة
    ├── js/ui.js                20 شاشة: رئيسية، أوضاع، فريق، بطولات، مهام، متجر، ملف،
    │                           أكاديمية، تدريب، مهارات، مجتمع آمن، متصدرين، إشعارات…
    ├── js/game-engine.js       محرك المباراة (أعيدت كتابته — انظر أدناه)
    └── js/main.js              الحالة، التنقل، الجويستيك، الأزرار، الخريطة المصغرة، PWA
```

وثائق الحزمة الأصلية (المعمارية، خريطة الشاشات، خطة الإنتاج، سياسة الأصول النظيفة،
نظام الصوت العربي، قوائم الفحص) في `games/football-future/docs/`.

## الجرافيكس — عرض بمنظور كاميرا البث (المحرك v4)

نظام العرض مبني بأسلوب ألعاب كرة القدم الحديثة (محاكاة تقديم FC) — لا منظور علوي ولا أيقونات:

- **منظور بث تلفزيوني**: الملعب شبه منحرف بإسقاط منظوري حقيقي (الخطوط، الدوائر،
  مناطق الجزاء كلها مُسقطة)، و**كاميرا تتبع الكرة** بنعومة عبر الملعب.
- **لاعبون مجسّمون**: شخصيات بشرية كاملة — رأس وشعر، قميص بتدرج وشريط الهوية المائل
  ورقم اللاعب، شورت، أرجل وأذرع بحركة جري متزامنة مع السرعة — يكبرون كلما اقتربوا
  من الكاميرا. أطقم واقعية: الفريق كحلي داكن بقطع ليموني، الخصم أبيض/أحمر.
- **استاد بانوراما**: مدرج بعيد بطبقات وجمهور كثيف بلمسات ليموني/سماوي (بارالاكس مع
  الكاميرا)، سقف بخط توهج، أعمدة إنارة، وشريط **لوحات LED** بعبارات الهوية.
- **مرميان ثلاثيا الأبعاد**: قائمان وعارضة بارتفاع منظوري صحيح لكل عمق، شباك جانبية
  وخلفية بنسيج شبكي، وتوهج بلوني الهوية (سماوي/ليموني).
- **كرة بقوس طيران**: ارتفاع حقيقي عند التسديد والتمرير مع ظل يتقلص في الهواء،
  نقش خماسي دوّار، وذيل توهج.
- **مؤثرات سينمائية**: قصاصات احتفال + وميض + اهتزاز كاميرا عند الأهداف، حلقات طاقة
  للمهارات، شعار FF باهت في دائرة المنتصف، ووميض جمهور حي.

## ترقية أسلوب اللعب

- حارسا مرمى بذكاء تصدٍّ (يتحرك بمحاذاة الكرة ويتصدى للتسديدات القوية).
- الكرات العالية لا تُلتقط أثناء الطيران — التوقيت جزء من اللعب.
- شوطان مع استراحة منتصف، وضربة بداية صحيحة بعد كل هدف (إعادة تموضع الفريقين).
- تبديل اللاعب: زر التمرير يبدّل لأقرب لاعب عند الدفاع، وتبديل تلقائي عند الاستلام.
- زر التسديد يتحول لضغط/افتكاك عند عدم الاستحواذ.
- ذكاء خصم متوازن للأعمار المستهدفة (ضغط بأقرب مدافعين، تمرير تحت الضغط، تسديد في المدى).
- دعم لوحة المفاتيح: الأسهم/WASD حركة، `X` تسديد، `Z` تمرير، `C` مهارة، `V` ضغط، `Shift` ركض.
- إحصاءات المباراة (تسديدات/تمريرات/افتكاكات) تغذي نظام المهام والتقدم.

## أمان الأطفال

بلا إعلانات، بلا مشتريات حقيقية (المتجر تجريبي بعملات اللعبة)، تواصل اجتماعي بردود جاهزة
فقط، والتقدم محفوظ محلياً على الجهاز (`localStorage`).

## التشغيل محلياً

```bash
npm run dev            # ثم http://localhost:3000/football-future
# أو مباشرة (ملفات ثابتة):
python3 -m http.server 8000 --directory public
# ثم http://localhost:8000/football-future/index.html
```

للفحص السريع: `?dur=N` يقصّر مدة المباراة بالثواني (للاختبار فقط).
