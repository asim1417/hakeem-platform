# أفضل الممارسات العالمية المعتمدة في هذه الحزمة

## 1. أصلية الهوية

- الشعار مرسوم كـ SVG أصلي.
- الأيقونات داخل الواجهة نصية/SVG/رمزية أصلية.
- لا توجد صور أو خطوط أو مكتبات خارجية ملزمة.
- لا توجد أسماء علامات ألعاب قائمة داخل الكود.

## 2. قابلية التشغيل

- لا توجد dependencies إلزامية.
- يمكن فتح `index.html` مباشرة.
- يمكن تشغيل سيرفر محلي عبر Python.
- يوجد Smoke Test.
- يوجد Validator.

## 3. فصل المسؤوليات

- `data.js`: البيانات والنصوص.
- `ui.js`: الواجهة والشاشات.
- `game-engine.js`: منطق اللعب والرسم.
- `main.js`: الربط والتحكم والحالة.

## 4. ملاءمة الأطفال 7–14

- أزرار كبيرة.
- تباين عالٍ.
- نصوص عربية مباشرة.
- تعليمات داخل المباراة.
- تحكم مبسط.
- متجر تجريبي مع توضيح ضرورة موافقة ولي الأمر في الإنتاج.

## 5. الأداء

- Canvas واحد للمباراة.
- DOM للشاشات فقط.
- CSS متغيرات Tokens.
- لا تحميل شبكي خارجي.
- Service Worker للتخزين المؤقت عند التشغيل عبر HTTP.

## 6. قابلية التحويل لمحرك احترافي

النموذج الحالي يعزل الأنظمة بحيث يسهل نقلها إلى:

- Unity UI Toolkit / Canvas.
- Godot Control Nodes.
- React Native.
- React + Phaser.
- Unreal UMG.

## 7. جودة الواجهة

- Glass panels.
- Gradients.
- Motion cues.
- Bottom navigation.
- Cards system.
- Progress bars.
- Badges.
- HUD.

## 8. اختبار سريع قبل التسليم

نفذ:

```bash
python3 smoke_test.py
python3 tools/validate_project.py
```

ثم افتح:

```bash
python3 run_local.py
```
