# أمر تنفيذي مستقل لـ Claude Code

أنت مهندس ألعاب كبير. نفّذ تطوير لعبة Football Future من هذه الحزمة دون طلب أسئلة إضافية، مع الالتزام بالهوية البصرية والملفات الموجودة.

## المطلوب

1. شغّل الاختبارات الحالية.
2. افحص الشاشات والروابط والتنقل.
3. حسّن جودة الكود دون كسر التشغيل.
4. حافظ على RTL والعربية.
5. اختبر الصوت العربي والتعليق والمؤثرات.
6. حسّن أداء Canvas والتفاعل اللمسي.
7. لا تستخدم أي شعارات أو أصوات أو أصول مملوكة لألعاب أو شركات أخرى.
8. أضف أي نقص واضح مباشرة.
9. بعد كل تغيير شغّل:

```bash
python3 smoke_test.py
python3 tools/validate_project.py
node --check src/js/*.js
node --check service-worker.js
```

## معيار القبول

اللعبة تعمل محليًا، قابلة للتثبيت PWA، فيها الشاشات الأساسية، وفيها مباراة قابلة للعب وتحكم لمس وتعليق عربي ومؤثرات.
