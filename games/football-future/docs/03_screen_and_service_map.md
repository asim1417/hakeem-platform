# خريطة الشاشات والخدمات

## الشاشات الرئيسية

| الشاشة | الوظيفة | الملف/الدالة |
|---|---|---|
| الرئيسية | مركز اللاعب والأحداث السريعة | `FFUI.home` |
| أوضاع اللعب | اختيار نمط المباراة أو التدريب | `FFUI.modes` |
| إدارة الفريق | التشكيلة واللاعبون والتكتيك | `FFUI.team` |
| البطولات | كأس المستقبل والبطولات القادمة | `FFUI.tournaments` |
| المهام | مهام يومية وأسبوعية ومكافآت | `FFUI.missions` |
| المتجر | باقات وعملات وعناصر | `FFUI.shop` |
| الملف الشخصي | هوية اللاعب والإحصاءات | `FFUI.profile` |
| الإعدادات | الصوت واللغة والأمان | `FFUI.settings` |
| قبل المباراة | التشكيلة قبل اللعب | `FFUI.preMatch` |
| اللعب المباشر | المباراة والتحكم | `FFUI.liveMatch` + `FootballFutureEngine` |
| ملخص المباراة | النتيجة والمكافآت | `FFUI.matchSummary` |

## الخدمات المطلوبة عند الانتقال للإنتاج

| الخدمة | الغرض |
|---|---|
| Player Account | حساب اللاعب وحفظ التقدم |
| Avatar | تخصيص شخصية الطفل |
| Squad | إدارة التشكيلة والبطاقات |
| Match Engine | محرك المباراة |
| Missions | المهام والمكافآت |
| Progression | XP، مستويات، شجرة مهارات |
| Store | متجر آمن مع موافقة ولي الأمر |
| Inventory | العناصر والملابس والكرات |
| Tournaments | بطولات وجداول ومكافآت |
| Leaderboard | ترتيب عالمي/أصدقاء |
| Safe Social | أصدقاء ونادٍ وتواصل آمن |
| Parental Safety | ضوابط ولي الأمر |

## تدفق اللعب الأساسي

```text
الرئيسية → اختيار نمط اللعب → قبل المباراة → اللعب المباشر → ملخص المباراة → المهام/المكافآت → الرئيسية
```

## تدفق التقدم

```text
الرئيسية → الملف الشخصي → تدريب/مهام → XP → تطوير اللاعب → فريق أقوى → بطولات
```

## تدفق الخدمات الحية

```text
الرئيسية → أحداث موسمية → بطولة → مكافآت → متجر/تخصيص → تقدم
```
