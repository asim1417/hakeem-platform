# أمر Claude Code للتنفيذ الذاتي

انسخ الأمر التالي داخل Claude Code بعد رفع هذا المجلد كاملاً:

```text
أنت الآن مهندس ألعاب أول، ومهندس واجهات، ومهندس جودة، ومهندس تجربة مستخدم للأطفال.

المشروع: Football Future / فوتبول فيوتشر.
الهدف: تطوير النموذج الحالي إلى نسخة لعبة جوال/Web احترافية، مع الحفاظ الكامل على الهوية البصرية الموجودة في visual_boards وعلى الكود الحالي.

التعليمات التنفيذية:

1. افحص المشروع كاملاً قبل أي تعديل:
   - README_FIRST.md
   - INSTALL_RUN_AR.md
   - docs/*.md
   - src/css/*.css
   - src/js/*.js
   - visual_boards/*.png

2. شغّل الاختبارات الحالية:
   - python3 smoke_test.py
   - python3 tools/validate_project.py

3. لا تغيّر الهوية البصرية الأساسية:
   - الخلفية: Deep Black / Graphite / Deep Navy.
   - الألوان: Electric Lime / Cyan / Teal / White / Silver.
   - الاتجاه: Arabic-first RTL.
   - الأسلوب: لعبة كرة قدم مستقبلية، أصلية، آمنة للأطفال 7–14.

4. طور المشروع دون طلب موافقات جزئية. اتخذ قرارات هندسية مناسبة وفق أفضل الممارسات.

5. المطلوب في المرحلة التالية:
   أ. تحسين محرك المباراة:
      - تمرير أذكى.
      - ذكاء اصطناعي أفضل للخصم.
      - حارس مرمى.
      - أخطاء وركلات حرة.
      - إعادة لقطات مبسطة.
      - ملخص إحصائي أدق.

   ب. تحسين الواجهة:
      - إضافة شاشة تدريب.
      - إضافة شجرة مهارات.
      - إضافة تخصيص لاعب.
      - إضافة لوحة أحداث موسمية.
      - إضافة Leaderboard تجريبي.

   ج. تحسين الجودة:
      - لا تستخدم أي أصل بصري أو اسم أو شعار تابع لعلامة خارجية.
      - حافظ على قابلية التشغيل بدون إنترنت.
      - حافظ على سهولة القراءة للأطفال.
      - لا تكسر التشغيل الحالي.

6. بعد كل مجموعة تغييرات:
   - شغّل smoke_test.py.
   - شغّل validate_project.py.
   - افحص index.html في المتصفح أو عبر سيرفر محلي.
   - أصلح أي خطأ قبل الانتقال للخطوة التالية.

7. عند الانتهاء سلمني:
   - ملخص التغييرات.
   - الملفات المعدلة.
   - طريقة التشغيل.
   - قائمة ما تم اختباره.
   - قائمة المرحلة التالية.

ممنوع:
- استخدام أسماء أو شعارات أو واجهات محمية لألعاب قائمة.
- إضافة dependencies بدون ضرورة.
- كسر دعم RTL.
- جعل الأزرار أصغر من 44px.
- ترك أخطاء Console.

ابدأ الآن بتنفيذ التحسينات مباشرة.
```
