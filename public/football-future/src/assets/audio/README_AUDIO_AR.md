# نظام الأصوات العربية والمؤثرات

هذه الحزمة لا تحتوي على أي تسجيلات صوتية خارجية أو مؤثرات من ألعاب أو مكتبات تجارية.

## ما الموجود فعليًا؟
1. مؤثرات صوتية أصلية مولدة بالكود عبر Web Audio API: نقر، انتقال، تمرير، تسديد، هدف، مكافأة، خطأ، صفارة، جمهور.
2. تعليقات عربية مولدة وقت التشغيل عبر Web Speech API/SpeechSynthesis بلغة `ar-SA` متى كان المتصفح/الجهاز يدعم صوتًا عربيًا.
3. بديل آمن عند عدم توفر الصوت العربي: مؤثرات صوتية مولدة بالكود دون تحميل ملفات خارجية.

## لماذا هذا آمن حقوقيًا؟
- لا توجد ملفات MP3/WAV مأخوذة من الإنترنت.
- لا توجد أصوات معلّقين أو علامات تجارية أو شعارات صوتية مملوكة.
- جميع العبارات العربية أصلية داخل `voice_ar_manifest.json`.

## للتطوير الإنتاجي لاحقًا
يمكن استبدال SpeechSynthesis بتسجيلات عربية أصلية من معلق/ممثل صوتي بعقد تنازل واضح، أو TTS تجاري بترخيص مكتوب يسمح بالاستخدام داخل لعبة.
