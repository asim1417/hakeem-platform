- **الربا والصرف** → نظام المعاملات المدنية — أحكام العقد > محل العقد ومشروعيته (مع مراعاة منع الربا)
- **السَّلَم والاستصناع** → نظام المعاملات المدنية — العقود الناقلة للملكية > البيع / الاستصناع
- **الخيارات** → نظام المعاملات المدنية — أحكام العقد > الخيارات وفسخ العقد والعيوب
- **المرابحة والتولية** → نظام المعاملات المدنية — العقود الناقلة للملكية > عقد البيع (المرابحة)
- **الإجارة** → نظام المعاملات المدنية — العقود المسماة > عقد الإيجار
- **الشركة** → نظام الشركات / نظام المعاملات المدنية — عقود المشاركات > عقد الشركة
- **المضاربة (القِراض)** → نظام المعاملات المدنية — عقود المشاركات > عقد المضاربة
- **المزارعة والمساقاة** → نظام المعاملات المدنية — عقود المشاركات > عقد المشاركة في الناتج (المزارعة/المساقاة)
- **الرهن** → نظام المعاملات المدنية — الحقوق العينية التبعية (التأمينات) > الرهن
- **الضمان والكفالة** → نظام المعاملات المدنية — العقود المسماة > عقد الكفالة
- **الحوالة** → نظام المعاملات المدنية — انتقال الالتزام > حوالة الحق وحوالة الدين
- **الوكالة** → نظام المعاملات المدنية — العقود الواردة على العمل > عقد الوكالة
- **الصلح** → نظام المعاملات المدنية — العقود الناقلة للملكية > عقد الصلح
- **الشُّفعة** → نظام المعاملات المدنية — الحقوق العينية الأصلية > الملكية > الشفعة
- **الهبة** → نظام المعاملات المدنية — العقود الناقلة للملكية > عقد الهبة
- **العارية** → نظام المعاملات المدنية — العقود المسماة > العارية / الانتفاع
- **الوديعة** → نظام المعاملات المدنية — العقود الواردة على العمل > عقد الإيداع
- **الغصب والإتلاف** → نظام المعاملات المدنية — مصادر الالتزام > الفعل الضار (الضرر والتعويض)
- **اللقطة واللقيط** → نظام المعاملات المدنية / الأحوال الشخصية — الفعل النافع (الإثراء بلا سبب) + أحكام اللقيط
- **إحياء الموات** → نظام المعاملات المدنية — الحقوق العينية الأصلية > الملكية وأسباب كسبها
- **الوقف** → أنظمة الأوقاف — أحكام الوقف (ترتيبات الهيئة العامة للأوقاف)
- **القِسمة** → نظام المعاملات المدنية — الحقوق العينية > الملكية الشائعة والقسمة
- **الجَعالة والسَّبق** → نظام المعاملات المدنية — مصادر الالتزام > الإرادة المنفردة (الوعد بجائزة/الجعالة)
- **القرض** → نظام المعاملات المدنية — العقود الناقلة للملكية > عقد القرض
- **التفليس والحجر** → نظام الإفلاس / نظام المعاملات المدنية — الأشخاص والأهلية (الحجر) + إجراءات الإفلاس
- **بيوع ومعاملات معاصرة** → أنظمة خاصة + المعاملات المدنية — العقود غير المسماة + أنظمة (الأوراق التجارية/التمويل/التأمين)
- **النكاح** → نظام الأحوال الشخصية — الباب الأول: الزواج (الأركان والشروط)

- **العشرة وحقوق الزوجين** → نظام الأحوال الشخصية — الباب الأول: الزواج (حقوق الزوجين)
- **الطلاق** → نظام الأحوال الشخصية — الباب الثالث: الفرقة بين الزوجين > الطلاق
- **الخلع** → نظام الأحوال الشخصية — الباب الثالث: الفرقة بين الزوجين > الخلع
- **الرجعة** → نظام الأحوال الشخصية — الباب الثالث: الفرقة بين الزوجين (الرجعة)
- **الإيلاء والظهار** → نظام الأحوال الشخصية — الباب الثالث: الفرقة (فسخ الزواج وأسبابه)
- **اللعان** → نظام الأحوال الشخصية — الباب الثاني/الثالث: النسب واللعان
- **العِدّة والاستبراء** → نظام الأحوال الشخصية — الباب الرابع: آثار الفرقة > العدة
- **الرضاع** → نظام الأحوال الشخصية — الباب الثاني: آثار عقد الزواج > النسب/الرضاع
- **النفقات** → نظام الأحوال الشخصية — الباب الثاني: آثار عقد الزواج > النفقة
- **الحضانة** → نظام الأحوال الشخصية — الباب الرابع: آثار الفرقة > الحضانة
- **النسب** → نظام الأحوال الشخصية — الباب الثاني: آثار عقد الزواج > النسب
- **الفرائض (المواريث)** → نظام الأحوال الشخصية — الباب السابع: التركة والإرث
- **الوصايا** → نظام الأحوال الشخصية — الباب السادس: الوصية
- **الجنايات والقصاص** → أحكام شرعية + أنظمة جزائية — القصاص (أحكام شرعية) + نظام الإجراءات الجزائية
- **الديات** → أحكام شرعية + أنظمة — الديات (أحكام شرعية) + اللائحة/الجداول المعتمدة
- **القسامة** → أحكام شرعية — القسامة (أحكام شرعية قضائية)
- **حد الزنا** → أحكام شرعية (حدود) — حد الزنا (أحكام شرعية)
- **حد القذف** → أحكام شرعية (حدود) — حد القذف (أحكام شرعية) + أنظمة الحماية من الإيذاء
- **حد السرقة** → أحكام شرعية (حدود) — حد السرقة (أحكام شرعية) + الأنظمة الجزائية
- **حد الحرابة** → أحكام شرعية (حدود) — الحرابة (أحكام شرعية) + أنظمة جزائية خاصة
- **حد الشرب والمسكر** → أحكام شرعية + أنظمة — المسكرات (أحكام شرعية) + نظام مكافحة المخدرات
- **الردة والبغي** → أحكام شرعية — الردة والبغي (أحكام شرعية قضائية)
- **التعزير** → أنظمة جزائية تعزيرية — العقوبات التعزيرية (الأنظمة الجزائية والمرجعية القضائية)
- **القضاء** → نظام المرافعات الشرعية — الاختصاص والولاية القضائية
- **الدعوى** → نظام المرافعات الشرعية — رفع الدعوى وقيدها وشروط قبولها
- **البينات والقرائن** → نظام الإثبات — أدلة الإثبات والقرائن
- **الشهادة** → نظام الإثبات — الشهادة (الباب الخاص بالشهادة)
- **اليمين والنكول** → نظام الإثبات — اليمين (الباب الخاص باليمين)
- **الإقرار** → نظام الإثبات — الإقرار (الباب الخاص بالإقرار)
- **التحكيم والصلح القضائي** → نظام التحكيم / المرافعات الشرعية — التحكيم + الصلح القضائي
- **الأيمان والنذور والكفارات** → أحكام شرعية عامة — خارج التقنين النظامي (مرجعية شرعية)
